# Computer Vision Project

## Важное примечание о файлах

### Файлы, не включённые в репозиторий

Некоторые файлы не были добавлены в репозиторий из-за ограничений GitHub на размер файлов (максимум 100 МБ):

- **Модели**: `*.keras`, `*.h5` (файлы моделей весят ~290 МБ)
- **Изображения датасета**: (размер набора данных превышает лимиты GitHub)

### Где найти эти файлы

1. **Модели**:
   - Обученные модели в формате `.keras` и `.h5`
   - Для работы с проектом необходимо обучить модель заново 

2. **Датасет**:
   - Набор изображений для обучения
   - Скачать датасеты можно по [ссылке](https://www.kaggle.com/datasets/icebearogo/fruit-classification-dataset) и [ссылке](https://www.kaggle.com/datasets/antobenedetti/animals)
